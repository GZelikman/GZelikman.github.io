Gym Diet Helper

Keywords: Python Tkinter Project

The Code of this project is on Github

The Idea came for this project, because one of my Hobbies is powerlifting and bodybuilding, the combination is called powerbuilding
So my Idea was to have an application, in which I can track my weight, calories and more.

In the end I created a program which takes input of weight, calories and burned calories daily,
also it takes the diet at input, which could be bulking, cutting or maintaining.
From these datapoints it gives tips for continuing your diet and also changes the daily calorie intake for you diet,
depending on your metrics. As an example if you are cutting and your calorie intake is under you given daily calorie intake,
but you don't lose weight, it will reduce the needed daily calorie intake.

I first created an terminal based program, which work pretty nicely but isn't really comftable to use
So I tried customtkinter for the first time to create a GUI for the program. In the end I am happy how it turned out,
except I needed to use dialog boxes to get input and to show info, with that I am not happy and will probably change it someday.

As you can see on the picture, there are some buttons with diffrent fuctions, which explains them selfs

For this project, saving the data is also an important feature,
so for this is used pickle for the first time, it is a great tool, to save Data locally.

Conclusion

It was a fun project, which I will continue to work on, to make it better and more user friendly.
The features are not that important right now, I would like to make the GUI better and more responsive and get rid of the dialog boxes.
